# MemberAvatar

Shows avatar of first character of member's (firstName, lastName or displayName) or if provided with photo url will show photo avatar.
Uses WHAvatar component

## Props

<Props of={MemberAvatar} />

## Examples

### First Name

<Playground>
  <MemberAvatar
    profile={{
      id: 2339,
      firstName: "John",
      picture: null
    }}
  />
</Playground>
